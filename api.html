    API Documentation¶

    Base¶

    base.Base() Base class for LISC analyses.

    Counts¶

    count.Count() This is a class for counting co-occurence of pre-specified terms list(s).

    Words¶

    words.Words() Class for searching through words in the abstracts of specified papers.

    Scrape¶

    scrape.scrape_counts(terms_lst_a[, …]) Search through pubmed for all abstracts for co-occurence.
    scrape.scrape_words(terms_lst[, …]) Search and scrape from pubmed for all abstracts referring to a given term.

    Data¶

    data.Data(label[, term]) An object to hold the word results for a given term or term.

    DataAll¶

    data_all.DataAll(term_data) Object to hold term data, aggregated across papers.
