    API Documentation¶

    The following is a list of the publicly available objects and functions in LISC.

    Note that many of the elements listed here are objects, indicated by CamelCase.

    If you click on the object names, it will take you to a new page describing their attributes and methods.

    Objects¶

    Counts() A class for collecting and analyzing co-occurence data of pre-specified terms list(s).
    Words() Class for searching through words in the abstracts of specified papers.

    Collect¶

    collect_info([db, api_key, logging, folder, …]) Collect database information & metadata from pubmed.
    collect_counts(terms_a[, inclusions_a, …]) Collect word co-occurence data from pubmed.
    collect_words(terms[, inclusions, …]) Collect text data and metadata from pubmed using specified search term(s).
    collect_citations(dois[, util, logging, …]) Scape OpenCitations for citation data.

    Data Objects¶

    Term Search term definition with inclusion & exclusion words.
    Data(term) An object to hold the word results for a given term.
    DataAll(term_data[, exclusions]) Object to hold term data, aggregated across papers.
    MetaData() An object to hold the meta data for data collection.

    Requester¶

    Requester([wait_time, logging, folder]) Object to handle URL requests.

    URLS¶

    URLs(base[, utils, authenticated]) Class to hold URL information for an API interface.
    EUtils([db, retmax, field, retmode, …]) Class to hold URLs for the NCBI EUtils API.
    OpenCitations() Class to hold URLs for the OpenCitations API.

    Plotting¶

    single.plot_years(years[, year_range, ax]) Plot publications across years histogram.
    group.plot_matrix(data[, x_labels, …]) Plot the matrix of percent asscociations between terms.
    group.plot_clustermap(data[, x_labels, …]) Plot clustermap.
    group.plot_dendrogram(data[, labels, ax]) Plot dendrogram.
    wordcloud.plot_wordcloud(freq_dist, n_words) Create and display a wordcloud.

    Database Management¶

    SCDB([base_path, auto_gen]) Database object for a SCANR project.
    create_file_structure([base_path, dir_name]) Create the file structure for a SCANR database.

    Utilities¶

    save_object(obj, f_name[, folder]) Save a custom object from LISC as a pickle file.
    load_object(f_name[, folder]) Load a custom object, from a pickle file.
