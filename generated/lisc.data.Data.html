    check_results()[source]¶

    Check for consistencty in extracted results.

    If everything worked, each data field (ids, titles, words, years) should have the same length, equal to the number of articles. Some entries may be blank (missing data), but if the lengths are not the same then the data does not line up and cannot be trusted.
