    Initialize the ncbi e-utils urls, with provided settings.

    Parameters:
    • db (str, optional) – Which database to access from EUtils.
    • retmax (int, optional) – The maximum number of articles to return.
    • field (str, optional) – The search field to search within.
    • retmode ({'lxml', 'xml'}, optional) – The return format for the results.
    • usehistory ({'n', 'y'}) – Whether to use history caching on the EUtils server.
    • api_key (str, optional) – An API key for authenticated NCBI user account.
