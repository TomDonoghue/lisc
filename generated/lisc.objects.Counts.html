    lisc.objects.Counts¶

    class lisc.objects.Counts[source]¶

    A class for collecting and analyzing co-occurrence data for specified terms list(s).

    terms¶

    Search terms to use.

    Type:dict
    counts¶

    The numbers of articles found for each combination of terms.

    Type:2d array
    score¶

    A transformed ‘score’ of co-occurrence data This can be a normalized version of the data, and/or a computed association index.

    Type:2d array
    square¶

    Whether the count data matrix is symmetrical.

    Type:bool
    meta_data¶

    Meta data information about the data collection.

    Type:MetaData object
