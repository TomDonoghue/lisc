    lisc.utils.SCDB¶

    class lisc.utils.SCDB(base=None, generate_paths=True, structure={1: {'base': ['terms', 'logs', 'data', 'figures']}, 2: {'data': ['counts', 'words']}, 3: {'words': ['raw', 'summary']}})[source]¶

    Database object for a SCANR project.

    Notes

    The default set of paths for SCDB is:

    • terms Term files.

    • logs Log files.

    • figs Figure files.

    • data Data files.

      • counts Counts data files.

      • words Words data files.

        • raw Raw words data files

        • summary Summary files for words data.
