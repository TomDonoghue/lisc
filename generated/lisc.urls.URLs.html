    authenticate(url)[source]¶

    Method to authenticate a URL for a given API.

    Parameters:
    url : str

    URL to add authentification to.
    Returns:
    str

    Authenticated URL.

    Notes

    This is a placeholder method, on the base URLs object, and should be overloaded by any API object that has authentification, with a method that implements whatever is needed to authenticate the URL for the API.
